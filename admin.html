📖 How To Use

🚀 Deploy on GitHub Pages (FREE)

Step 1 — Go to github.com → Sign Up (free)
Step 2 — New repository → name it evander-gas → Public
Step 3 — Upload index.html, admin.html, epag-logo.png, and all 5 cylinder images
Step 4 — Settings → Pages → select main branch → Save
Step 5 — Your live links:
   • Customers: https://yourusername.github.io/evander-gas/
   • Admin: https://yourusername.github.io/evander-gas/admin.html
